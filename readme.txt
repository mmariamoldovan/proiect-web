CleanPro-Website servicii curatenie

Descriere: 
Website pentru o companie de servicii de curatenie profesionala. Prezinta serviciile oferite, preturi si forumlar de contact.

Pagini:
-home.html - Pagina principala cu prezentare companie
-servicii.html - Lista servicii cu functie de cautare
-preturi.html - Formular de contact

Tehnologii:

-HTML5
-CSS3
-JavaScript

Functionalitati:
-Filtrare servicii dupa nume
-Calculator de pret cu reducere automata de 10%
-Formular de contact cu validare email si campuri
-Buton "Back to top" pentru scroll eficient
-Meniu navigare cu pagina activa evidentiata
-Design resposnive si intuitiv

Cum se ruleaza:
1.Deschide fisierul home.html intr-un browser 
sau
2.Pune proiectul in folderul htdocs din XAMPP si acceseaza localhost

Structura fisiere:
-home.html-Pagina acasa 
-servicii.html-Pagina servicii
-preturi.html-Pagina preturi
-contact.html-Pagina contact
-style.css-Toate stilurile CSS
-script.js-Toate functiile js
